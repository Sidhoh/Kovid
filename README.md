# Kovid V0.1.0
A python api to get info on covid-19

# Installation and Usage
To install the api:

```
pip install git+git://github.com/roooof/Kovid.git
```

To import the api:

```python
import kovid
```
Usage:

```python
# To get global data
kovid.get_global_data()

# To get data from all the countries
kovid.get_countries_data()
```
